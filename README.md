# Personal Academic Website

A modern, responsive, and elegant personal website template designed for academic researchers.

## Features
*   **Single Page Design**: Clean navigation between Home, Research, Publications, Teaching, and Contact.
*   **Dark Mode**: Automatic detection and manual toggle.
*   **Responsive**: Works on mobile and desktop.
*   **Easy to Customize**: Edit `index.html` for content and `assets/css/style.css` for colors.

## Installation
1.  Clone the repo.
2.  Open `index.html` in a browser.

## Deployment
Hosted on GitHub Pages.
